# AI Knowledge Platform 项目亮点

- **智能检索**：针对专有名词漏召回与多轮追问召回失效，在浏览器侧实现向量、中文关键词双路检索及 RRF 融合；基于质量阈值自动降级通用回答，并结合对话历史改写指代追问。

- **流式交互**：手写 SSE 解析与停止生成，处理中文字符和事件被数据块截断的问题；通过 rAF 合帧和 Markdown 增量渲染减少长回答重复解析，3.9 万字符、164 次切分累计 13ms。

- **向量缓存**：设计 IndexedDB 向量缓存，利用切片不可变的特点按 id 比对增量同步，缓存命中后单轮问答传输由约 3MB 降至 KB 级，不可用时自动回退全量拉取。

- **计算卸载**：将双路检索的评分计算移入 Web Worker；数百条向量合并为单个 Float32Array 整体移交，替代逐条深拷贝；实测 2000 条切片约 9ms，为知识库增长预留主线程余量。

- **E2E 测试**：基于 Playwright 实现后端接口 Mock，不依赖外部跑通 E2E 测试，保障核心业务流随时可验证。
