# AI Knowledge Platform 项目亮点

- **流式交互**：基于 fetch + ReadableStream 手写 OpenAI 兼容 SSE 解析，处理 UTF-8 字符及事件边界跨数据块；通过 AbortController 支持停止生成，并采用 rAF 合帧、Markdown 稳定段/尾段增量渲染及 `<think>` 分流减少重复解析，3.9 万字符、164 次切分累计 13ms。

- **引用溯源**：宽容解析模型输出的参考片段编号并渲染为可点击角标，兼容全角/半角、重复及越界引用；联动来源面板展开、平滑定位与闪烁高亮，形成回答到资料核验的交互闭环。

- **向量缓存**：基于切片不可变约束，以服务端与 IndexedDB 的 ID 集合 diff 实现新增切片增量拉取和失效数据淘汰；embedding 使用 Float32Array 存储，缓存命中后单轮传输由约 3MB 降至 KB 级，不可用时自动回退全量请求。

- **计算卸载**：将向量相似度与中文关键词评分移入 Web Worker，数百条向量打包为连续 Float32Array 矩阵，通过 Transferable 移交所有权，替代逐条结构化克隆；支持并发请求关联及 Worker 异常回退，实测 1536 维、2000 条切片计算约 9ms。

- **工程质量**：使用 Vitest 覆盖流式解析、Markdown 切分、`<think>` 状态机与引用解析等核心纯函数；通过 Playwright 路由拦截 Mock Supabase PostgREST 和 SSE 流，无真实后端与 AI Key 即可跑通登录、提问到流式回答的关键链路，并接入 GitHub Actions 持续验证。
