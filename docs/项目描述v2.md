### AI Knowledge Platform 个人知识库问答平台（RAG） | 独立开发 | 2026/03 - 2026/07

- **项目链接**：（部署地址待填）
- **技术栈**：Vue3 + TypeScript + Vite + Pinia + Element Plus + Supabase + OpenAI-compatible API
- **项目介绍**：基于 Vue3 + Supabase 构建的个人知识库 AI 问答平台，支持文档/文件切片入库、双路检索增强问答（RAG）与引用溯源，覆盖 AI 写作助手与管理后台。
- **项目亮点**：
  - 实现向量 + 中文 N-gram 关键词双路混合检索，RRF（k=60）名次融合，召回不足自动降级通用回答；多轮指代追问经 LLM 改写再检索，2.5s 超时回退原句。

智能检索：浏览器侧实现向量与中文关键词双路检索，通过 RRF 融合语义匹配与专有名词命中结果；检索结果不可靠时切换通用回答，并结合对话历史补全含指代或信息省略的追问，提升多轮检索准确性。

智能检索：针对专有名词漏召回与多轮追问召回失效，在浏览器侧实现向量、中文关键词双路检索及 RRF 融合；基于质量阈值自动降级通用回答，并结合对话历史改写指代追问。




  - 手写 OpenAI 兼容 SSE 流式解析（fetch + ReadableStream），处理 UTF-8 解码与事件边界跨 chunk，支持 AbortController 停止生成；采用 rAF 合帧 + Markdown 稳定段/尾段增量渲染 + think 推理分流，3.9 万字符长回答切分实测累计仅 13ms。
